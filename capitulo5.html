Capítulo 5: Condicionais

5.1 Estrutura if

A estrutura condicional if permite que o programa execute diferentes blocos de código baseado em condições.

Fluxo do if
condição → True: Executa bloco → False: Pula bloco
# Estrutura básica do if idade = 18 if idade >= 18: print("Você é maior de idade") print("Pode votar e dirigir") print("Esta linha sempre executa") # Exemplo com input nota = float(input("Digite sua nota: ")) if nota >= 7.0: print("Parabéns! Você foi aprovado!") if nota < 7.0: print("Você precisa estudar mais.") # Condições mais complexas temperatura = 25 humidade = 60 if temperatura > 30 and humidade > 70: print("Dia muito quente e úmido") if temperatura < 15 or humidade < 30: print("Condições climáticas extremas")

Dica: A indentação (espaços) é obrigatória em Python. Use 4 espaços para cada nível de indentação.

5.2 Estrutura if-else

O else permite executar um bloco alternativo quando a condição do if é falsa.

Fluxo do if-else
condição → True: Bloco if → False: Bloco else
# Estrutura if-else básica idade = int(input("Digite sua idade: ")) if idade >= 18: print("Você é maior de idade") print("Pode votar, dirigir e trabalhar") else: print("Você é menor de idade") print(f"Faltam {18 - idade} anos para a maioridade") # Exemplo com números numero = int(input("Digite um número: ")) if numero % 2 == 0: print(f"{numero} é par") else: print(f"{numero} é ímpar") # Exemplo com strings senha = input("Digite a senha: ") if senha == "123456": print("Acesso liberado!") else: print("Senha incorreta!") print("Acesso negado.")

5.3 Estrutura if-elif-else

O elif (else if) permite testar múltiplas condições em sequência.

Fluxo do if-elif-else
if → elif → elif → ... → else (opcional)
# Sistema de notas nota = float(input("Digite a nota (0-10): ")) if nota >= 9.0: conceito = "A" status = "Excelente!" elif nota >= 8.0: conceito = "B" status = "Muito bom!" elif nota >= 7.0: conceito = "C" status = "Bom!" elif nota >= 6.0: conceito = "D" status = "Regular" else: conceito = "F" status = "Reprovado" print(f"Nota: {nota}") print(f"Conceito: {conceito}") print(f"Status: {status}") # Classificação de idade idade = int(input("Digite sua idade: ")) if idade < 0: print("Idade inválida!") elif idade <= 12: categoria = "Criança" elif idade <= 17: categoria = "Adolescente" elif idade <= 59: categoria = "Adulto" elif idade <= 100: categoria = "Idoso" else: categoria = "Idade muito avançada" print(f"Categoria: {categoria}")

5.4 Condicionais Aninhadas

Você pode colocar estruturas condicionais dentro de outras estruturas condicionais.

# Exemplo de condicionais aninhadas usuario = input("Digite o usuário: ") senha = input("Digite a senha: ") if usuario == "admin": if senha == "123456": print("Login de administrador realizado com sucesso!") print("Acesso total ao sistema") else: print("Senha de administrador incorreta!") else: if usuario == "user": if senha == "user123": print("Login de usuário realizado com sucesso!") print("Acesso limitado ao sistema") else: print("Senha de usuário incorreta!") else: print("Usuário não encontrado!") # Sistema de desconto valor_compra = float(input("Valor da compra: R$ ")) tem_cartao = input("Tem cartão fidelidade? (s/n): ").lower() if valor_compra >= 100: if tem_cartao == "s": desconto = 0.15 # 15% de desconto print("Desconto de 15% aplicado!") else: desconto = 0.10 # 10% de desconto print("Desconto de 10% aplicado!") else: if tem_cartao == "s": desconto = 0.05 # 5% de desconto print("Desconto de 5% aplicado!") else: desconto = 0 # Sem desconto print("Nenhum desconto aplicado.") valor_final = valor_compra * (1 - desconto) print(f"Valor final: R$ {valor_final:.2f}")

5.5 Operador Ternário

Uma forma concisa de escrever condicionais simples em uma única linha.

# Operador ternário - sintaxe # valor_se_verdadeiro if condição else valor_se_falso idade = 20 status = "Maior" if idade >= 18 else "Menor" print(f"Status: {status}") # Comparação com if-else tradicional # Forma tradicional: if idade >= 18: status = "Maior" else: status = "Menor" # Exemplos práticos numero = int(input("Digite um número: ")) paridade = "Par" if numero % 2 == 0 else "ímpar" print(f"O número {numero} é {paridade}") # Determinando o maior de dois números a = 10 b = 15 maior = a if a > b else b print(f"O maior número é: {maior}") # Com strings nome = input("Digite seu nome: ") saudacao = f"Olá, {nome}!" if nome else "Olá, visitante!" print(saudacao) # Aninhando operadores ternários (não recomendado para casos complexos) nota = 8.5 resultado = "A" if nota >= 9 else "B" if nota >= 8 else "C" if nota >= 7 else "D" print(f"Conceito: {resultado}")

Cuidado: Use o operador ternário apenas para condicionais simples. Para lógica complexa, prefira if-elif-else tradicional.

Resumo do Capítulo

  • if: Executa bloco se condição for verdadeira
  • else: Executa bloco alternativo
  • elif: Testa múltiplas condições em sequência
  • Indentação: Obrigatória para definir blocos de código
  • Condicionais aninhadas: if dentro de if
  • Operador ternário: condição em uma linha
  • Use operadores lógicos (and, or, not) para condições complexas
