Capítulo 4: Operadores

4.1 Operadores Aritméticos

Os operadores aritméticos são usados para realizar operações matemáticas básicas.

Operador Descrição Exemplo Resultado
+ Adição 5 + 3 8
- Subtração 5 - 3 2
* Multiplicação 5 * 3 15
/ Divisão 10 / 3 3.333...
// Divisão inteira 10 // 3 3
% Módulo (resto) 10 % 3 1
** Exponenciação 2 ** 3 8
# Exemplos práticos de operadores aritméticos a = 10 b = 3 print(f"a + b = {a + b}") # 13 print(f"a - b = {a - b}") # 7 print(f"a * b = {a * b}") # 30 print(f"a / b = {a / b}") # 3.333... print(f"a // b = {a // b}") # 3 print(f"a % b = {a % b}") # 1 print(f"a ** b = {a ** b}") # 1000 # Precedência de operadores resultado = 2 + 3 * 4 # 14 (não 20) print(f"2 + 3 * 4 = {resultado}") # Usando parênteses para alterar precedência resultado = (2 + 3) * 4 # 20 print(f"(2 + 3) * 4 = {resultado}")

4.2 Operadores de Comparação

Os operadores de comparação são usados para comparar valores e retornam True ou False.

Operador Descrição Exemplo Resultado
== Igual a 5 == 5 True
!= Diferente de 5 != 3 True
> Maior que 5 > 3 True
< Menor que 5 < 3 False
>= Maior ou igual 5 >= 5 True
<= Menor ou igual 3 <= 5 True
# Exemplos de operadores de comparação a = 10 b = 5 c = 10 print(f"a == b: {a == b}") # False print(f"a == c: {a == c}") # True print(f"a != b: {a != b}") # True print(f"a > b: {a > b}") # True print(f"a < b: {a < b}") # False print(f"a >= c: {a >= c}") # True print(f"b <= a: {b <= a}") # True # Comparação de strings nome1 = "Ana" nome2 = "Bruno" print(f"'{nome1}' < '{nome2}': {nome1 < nome2}") # True (ordem alfabética) # Comparação de tipos diferentes print(f"5 == '5': {5 == '5'}") # False (tipos diferentes)

4.3 Operadores Lógicos

Os operadores lógicos são usados para combinar expressões booleanas.

Operador Descrição Exemplo Resultado
and E lógico True and False False
or OU lógico True or False True
not NÃO lógico not True False
# Exemplos de operadores lógicos idade = 20 salario = 3000 # AND - ambas condições devem ser verdadeiras if idade >= 18 and salario >= 2000: print("Aprovado para o empréstimo") # OR - pelo menos uma condição deve ser verdadeira if idade < 18 or salario < 1000: print("Não aprovado") else: print("Análise necessária") # NOT - inverte o valor booleano tem_carteira = True if not tem_carteira: print("Não pode dirigir") else: print("Pode dirigir") # Combinando operadores if (idade >= 18 and idade <= 65) and (salario >= 2000 or tem_carteira): print("Perfil aprovado")

4.4 Operadores de Atribuição

Os operadores de atribuição são usados para atribuir valores às variáveis.

Operador Equivalente Exemplo Resultado
= - x = 5 x = 5
+= x = x + y x += 3 x = x + 3
-= x = x - y x -= 2 x = x - 2
*= x = x * y x *= 4 x = x * 4
/= x = x / y x /= 2 x = x / 2
%= x = x % y x %= 3 x = x % 3
**= x = x ** y x **= 2 x = x ** 2
# Exemplos de operadores de atribuição x = 10 print(f"Valor inicial: {x}") x += 5 # x = x + 5 print(f"Após x += 5: {x}") # 15 x -= 3 # x = x - 3 print(f"Após x -= 3: {x}") # 12 x *= 2 # x = x * 2 print(f"Após x *= 2: {x}") # 24 x /= 4 # x = x / 4 print(f"Após x /= 4: {x}") # 6.0 x **= 2 # x = x ** 2 print(f"Após x **= 2: {x}") # 36.0 # Atribuição múltipla a, b, c = 1, 2, 3 print(f"a={a}, b={b}, c={c}") # Troca de valores a, b = b, a print(f"Após troca: a={a}, b={b}")

4.5 Operadores de Identidade e Pertencimento

Operadores especiais para verificar identidade e pertencimento.

# Operadores de identidade (is, is not) a = [1, 2, 3] b = [1, 2, 3] c = a print(f"a == b: {a == b}") # True (mesmo conteúdo) print(f"a is b: {a is b}") # False (objetos diferentes) print(f"a is c: {a is c}") # True (mesmo objeto) # Operadores de pertencimento (in, not in) lista = [1, 2, 3, 4, 5] print(f"3 in lista: {3 in lista}") # True print(f"6 not in lista: {6 not in lista}") # True frase = "Python é incrível" print(f"'Python' in frase: {'Python' in frase}") # True print(f"'Java' in frase: {'Java' in frase}") # False

Exercícios Práticos

Exercício 1: Calculadora Básica

Crie uma calculadora que peça dois números e uma operação (+, -, *, /) e exiba o resultado.

# Calculadora Básica print("=== Calculadora Básica ===") # Entrada de dados num1 = float(input("Digite o primeiro número: ")) operacao = input("Digite a operação (+, -, *, /): ") num2 = float(input("Digite o segundo número: ")) # Cálculo baseado na operação if operacao == "+": resultado = num1 + num2 elif operacao == "-": resultado = num1 - num2 elif operacao == "*": resultado = num1 * num2 elif operacao == "/": if num2 != 0: resultado = num1 / num2 else: print("Erro: Divisão por zero!") resultado = None else: print("Operação inválida!") resultado = None # Exibição do resultado if resultado is not None: print(f"\n{num1} {operacao} {num2} = {resultado}")

Exercício 2: Verificador de Números

Crie um programa que verifique se um número é par ou ímpar, positivo ou negativo, e se é múltiplo de 3.

# Verificador de Números print("=== Verificador de Números ===") # Entrada numero = int(input("Digite um número inteiro: ")) print(f"\nAnálise do número {numero}:") # Verificar par ou ímpar if numero % 2 == 0: print("É par") else: print("É ímpar") # Verificar positivo, negativo ou zero if numero > 0: print("É positivo") elif numero < 0: print("É negativo") else: print("É zero") # Verificar múltiplo de 3 if numero % 3 == 0: print("É múltiplo de 3") else: print("Não é múltiplo de 3") # Verificar se é múltiplo de 5 E de 3 if numero % 3 == 0 and numero % 5 == 0: print("É múltiplo de 3 E de 5") # Verificar se é múltiplo de 2 OU de 7 if numero % 2 == 0 or numero % 7 == 0: print("É múltiplo de 2 OU de 7")

Exercício 3: Sistema de Login

Crie um sistema simples que verifique se o usuário e senha estão corretos usando operadores lógicos.

# Sistema de Login print("=== Sistema de Login ===") # Credenciais corretas (em um sistema real, isso seria mais seguro) usuario_correto = "admin" senha_correta = "123456" max_tentativas = 3 tentativas = 0 while tentativas < max_tentativas: # Entrada de dados usuario = input("Usuário: ") senha = input("Senha: ") # Verificação usando operadores lógicos if usuario == usuario_correto and senha == senha_correta: print("\nLogin realizado com sucesso!") print("Bem-vindo ao sistema!") break else: tentativas += 1 restantes = max_tentativas - tentativas if tentativas < max_tentativas: if usuario != usuario_correto and senha != senha_correta: print(f"\nUsuário e senha incorretos. {restantes} tentativas restantes.") elif usuario != usuario_correto: print(f"\nUsuário incorreto. {restantes} tentativas restantes.") else: print(f"\nSenha incorreta. {restantes} tentativas restantes.") else: print("\nNúmero máximo de tentativas excedido. Acesso bloqueado!")

Resumo do Capítulo

  • Aritméticos: +, -, *, /, //, %, **
  • Comparação: ==, !=, >, <, >=, <=
  • Lógicos: and, or, not
  • Atribuição: =, +=, -=, *=, /=, %=, **=
  • Identidade: is, is not
  • Pertencimento: in, not in
  • Precedência de operadores: parênteses alteram a ordem
