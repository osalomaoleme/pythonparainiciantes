Capítulo 6: Laços de repetição

6.1 Laço while

O laço while executa um bloco de código repetidamente enquanto uma condição for verdadeira.

Fluxo do while
Verifica condição → True: Executa bloco → Volta para condição → False: Sai do loop
# Exemplo básico de while contador = 1 while contador <= 5: print(f"Contagem: {contador}") contador += 1 # Importante: incrementar para evitar loop infinito print("Loop finalizado!") # Exemplo com input do usuário senha_correta = "123456" senha = "" while senha != senha_correta: senha = input("Digite a senha: ") if senha != senha_correta: print("Senha incorreta! Tente novamente.") else: print("Senha correta! Acesso liberado.") # Somando números até o usuário digitar 0 soma = 0 numero = 1 # Inicializar com valor diferente de 0 print("Digite números para somar (0 para parar):") while numero != 0: numero = float(input("Número: ")) if numero != 0: soma += numero print(f"Soma total: {soma}")

Cuidado: Sempre certifique-se de que a condição do while eventualmente se torne falsa, senão você terá um loop infinito!

6.2 Laço for

O laço for é usado para iterar sobre sequências (listas, strings, ranges, etc.).

Fluxo do for
Para cada item na sequência → Executa bloco → Próximo item → Fim da sequência: Sai
# For com range() print("Contando de 1 a 5:") for i in range(1, 6): print(f"Número: {i}") # Range com diferentes parâmetros print("\nContando de 0 a 9:") for i in range(10): # range(10) = 0 a 9 print(i, end=" ") print("\n\nContando de 2 em 2:") for i in range(0, 11, 2): # início, fim, passo print(i, end=" ") print("\n\nContando regressivamente:") for i in range(10, 0, -1): print(i, end=" ") # For com strings print("\n\nIterando sobre string:") nome = "Python" for letra in nome: print(f"Letra: {letra}") # For com listas print("\nIterando sobre lista:") frutas = ["maçã", "banana", "laranja", "uva"] for fruta in frutas: print(f"Fruta: {fruta}") # Enumerate para obter índice e valor print("\nCom índices:") for indice, fruta in enumerate(frutas): print(f"{indice + 1}. {fruta}")

6.3 Comandos break e continue

break interrompe o loop completamente, enquanto continue pula para a próxima iteração.

# Exemplo com break print("Procurando o número 7:") for numero in range(1, 11): if numero == 7: print(f"Encontrei o {numero}! Parando a busca.") break print(f"Verificando: {numero}") print("\n" + "="*30) # Exemplo com continue print("Números ímpares de 1 a 10:") for numero in range(1, 11): if numero % 2 == 0: # Se for par continue # Pula para a próxima iteração print(numero) print("\n" + "="*30) # Exemplo prático: validação de entrada print("Digite números positivos (0 para sair):") while True: try: numero = float(input("Número: ")) if numero == 0: print("Saindo...") break if numero < 0: print("Número negativo ignorado.") continue print(f"Número válido: {numero}") print(f"Quadrado: {numero ** 2}") except ValueError: print("Entrada inválida! Digite apenas números.") continue

6.4 Loops Aninhados

Você pode colocar um loop dentro de outro loop para criar estruturas mais complexas.

# Tabuada completa print("=== TABUADA COMPLETA ===") for i in range(1, 11): print(f"\nTabuada do {i}:") for j in range(1, 11): resultado = i * j print(f"{i} x {j} = {resultado:2d}") print("\n" + "="*40) # Matriz de números print("Matriz 5x5:") for linha in range(5): for coluna in range(5): numero = linha * 5 + coluna + 1 print(f"{numero:3d}", end=" ") print() # Nova linha print("\n" + "="*40) # Padrão de asteriscos print("Triângulo de asteriscos:") for i in range(1, 6): for j in range(i): print("*", end="") print() # Nova linha print("\nTriângulo invertido:") for i in range(5, 0, -1): for j in range(i): print("*", end="") print() # Nova linha

6.5 Else em Loops

Python permite usar else com loops. O bloco else executa quando o loop termina normalmente (sem break).

# Else com for print("Procurando número par maior que 10:") numeros = [1, 3, 5, 7, 9, 11, 13] for numero in numeros: if numero > 10 and numero % 2 == 0: print(f"Encontrado: {numero}") break else: print("Nenhum número par maior que 10 foi encontrado.") print("\n" + "="*40) # Else com while print("Tentando adivinhar a senha:") senha_correta = "python123" tentativas = 0 max_tentativas = 3 while tentativas < max_tentativas: senha = input(f"Tentativa {tentativas + 1}: ") tentativas += 1 if senha == senha_correta: print("Senha correta! Acesso liberado.") break else: print("Senha incorreta.") else: print("Número máximo de tentativas excedido. Acesso bloqueado!")

Exercício 1: Contador de Números Pares

Crie um programa que conte quantos números pares existem entre 1 e 100, e calcule a soma desses números.

# Contador de Números Pares print("=== Contador de Números Pares (1-100) ===") contador_pares = 0 soma_pares = 0 lista_pares = [] # Método 1: usando for for numero in range(1, 101): if numero % 2 == 0: contador_pares += 1 soma_pares += numero lista_pares.append(numero) print(f"Quantidade de números pares: {contador_pares}") print(f"Soma dos números pares: {soma_pares}") print(f"Média dos números pares: {soma_pares / contador_pares:.2f}") # Exibindo os primeiros e últimos 5 números pares print(f"\nPrimeiros 5 pares: {lista_pares[:5]}") print(f"Últimos 5 pares: {lista_pares[-5:]}") # Método 2: usando range com passo 2 (mais eficiente) print("\n" + "="*40) print("Método alternativo (mais eficiente):") contador_pares_v2 = 0 soma_pares_v2 = 0 for numero in range(2, 101, 2): # Começa em 2, vai até 100, passo 2 contador_pares_v2 += 1 soma_pares_v2 += numero print(f"Quantidade: {contador_pares_v2}") print(f"Soma: {soma_pares_v2}")

")) if n <= 0: print("Por favor, digite um número positivo!") continue break except ValueError: print("Por favor, digite apenas números inteiros!") print(f"\nOs primeiros {n} números da sequência de Fibonacci:") print("="*50) # Casos especiais if n == 1: print("1. 0") elif n == 2: print("1. 0") print("2. 1") else: # Inicialização a, b = 0, 1 fibonacci_list = [a, b] print(f"1. {a}") print(f"2. {b}") # Gerar o restante da sequência for i in range(3, n + 1): proximo = a + b fibonacci_list.append(proximo) print(f"{i}. {proximo}") # Atualizar para próxima iteração a, b = b, proximo # Estatísticas da sequência if n > 1: print(f"\n{'='*50}") print("ESTATÍSTICAS:") print(f"Soma total: {sum(fibonacci_list)}") print(f"Maior número: {max(fibonacci_list)}") print(f"Números pares: {[num for num in fibonacci_list if num % 2 == 0]}") print(f"Números ímpares: {[num for num in fibonacci_list if num % 2 != 0]}") # Razão áurea (para sequências maiores) if n > 10: razao = fibonacci_list[-1] / fibonacci_list[-2] print(f"Razão entre os dois últimos números: {razao:.6f}") print("(Aproxima-se da razão áurea: 1.618034...)") print("\nSequência completa:", fibonacci_list)

Resumo do Capítulo

  • while: Repete enquanto condição for verdadeira
  • for: Itera sobre sequências (range, listas, strings)
  • range(): Gera sequências numéricas (início, fim, passo)
  • break: Sai do loop imediatamente
  • continue: Pula para próxima iteração
  • else em loops: Executa se loop terminar normalmente
  • enumerate(): Obtém índice e valor simultaneamente
  • Cuidado com loops infinitos - sempre altere a condição
