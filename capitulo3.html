Capítulo 3: Entrada e Saída de Dados

3.1 Função input()

A função input() é usada para receber dados do usuário. Ela sempre retorna uma string, mesmo que o usuário digite números.

# Exemplo básico de input nome = input("Digite seu nome: ") print(f"Olá, {nome}!") # Input com prompt personalizado idade = input("Qual é a sua idade? ") print(f"Você tem {idade} anos")

Importante: O input() sempre retorna uma string. Para usar números, você precisa converter usando int() ou float().

3.2 Conversão de Tipos

Para trabalhar com números vindos do input, precisamos converter os tipos de dados.

# Convertendo para inteiro idade = int(input("Digite sua idade: ")) print(f"No próximo ano você terá {idade + 1} anos") # Convertendo para float altura = float(input("Digite sua altura em metros: ")) print(f"Sua altura é {altura} metros") # Tratando possíveis erros try: numero = int(input("Digite um número: ")) print(f"O dobro do número é {numero * 2}") except ValueError: print("Erro: Digite apenas números!")

3.3 Função print() Avançada

A função print() tem vários parâmetros úteis para formatação de saída.

# Parâmetros do print() print("Olá", "mundo", sep="-") # Separador personalizado print("Primeira linha", end=" ") # Não quebra linha print("mesma linha") # Formatação com f-strings nome = "João" idade = 25 print(f"Nome: {nome}, Idade: {idade}") # Formatação de números preco = 19.99 print(f"Preço: R$ {preco:.2f}") # Alinhamento print(f"{'Nome':<10} {'Idade':>5}") print(f"{'João':<10} {25:>5}")

3.4 Formatação de Strings

Existem várias formas de formatar strings em Python.

# Método format() nome = "Maria" idade = 30 mensagem = "Olá, {}! Você tem {} anos.".format(nome, idade) print(mensagem) # Com índices mensagem = "Olá, {0}! Você tem {1} anos. {0} é um belo nome.".format(nome, idade) print(mensagem) # Com nomes mensagem = "Olá, {nome}! Você tem {idade} anos.".format(nome=nome, idade=idade) print(mensagem) # Formatação de números pi = 3.14159 print("Pi com 2 casas: {:.2f}".format(pi)) print("Pi em porcentagem: {:.1%}".format(pi/100))

Exercícios Práticos

Exercício 1: Calculadora de IMC

Crie um programa que peça o peso e altura do usuário e calcule o IMC (Índice de Massa Corporal). Fórmula: IMC = peso / (altura²)

# Calculadora de IMC print("=== Calculadora de IMC ===") # Entrada de dados peso = float(input("Digite seu peso (kg): ")) altura = float(input("Digite sua altura (m): ")) # Cálculo do IMC imc = peso / (altura ** 2) # Saída formatada print(f"\nSeu IMC é: {imc:.2f}") # Classificação if imc < 18.5: classificacao = "Abaixo do peso" elif imc < 25: classificacao = "Peso normal" elif imc < 30: classificacao = "Sobrepeso" else: classificacao = "Obesidade" print(f"Classificação: {classificacao}")

Exercício 2: Conversor de Temperatura

Crie um programa que converta temperatura de Celsius para Fahrenheit e Kelvin. Fórmulas: F = C × 9/5 + 32 | K = C + 273.15

# Conversor de Temperatura print("=== Conversor de Temperatura ===") # Entrada celsius = float(input("Digite a temperatura em Celsius: ")) # Conversões fahrenheit = celsius * 9/5 + 32 kelvin = celsius + 273.15 # Saída formatada print(f"\n{celsius}°C equivale a:") print(f"Fahrenheit: {fahrenheit:.1f}°F") print(f"Kelvin: {kelvin:.1f}K") # Tabela formatada print("\n" + "="*30) print(f"{'Celsius':<10} {'Fahrenheit':<12} {'Kelvin':<10}") print("="*30) print(f"{celsius:<10.1f} {fahrenheit:<12.1f} {kelvin:<10.1f}")

Exercício 3: Ficha Pessoal

Crie um programa que colete dados pessoais (nome, idade, cidade, profissão) e exiba uma ficha formatada.

# Ficha Pessoal print("=== Cadastro Pessoal ===") # Coleta de dados nome = input("Nome completo: ") idade = int(input("Idade: ")) cidade = input("Cidade: ") profissao = input("Profissão: ") email = input("E-mail: ") # Exibição da ficha print("\n" + "="*40) print(" FICHA PESSOAL") print("="*40) print(f"Nome: {nome}") print(f"Idade: {idade} anos") print(f"Cidade: {cidade}") print(f"Profissão: {profissao}") print(f"E-mail: {email}") print("="*40) # Mensagem personalizada if idade >= 18: status = "maior de idade" else: status = "menor de idade" print(f"\nOlá {nome.split()[0]}, você é {status} e mora em {cidade}.")

Resumo do Capítulo

  • A função input() sempre retorna string

  • F-strings são a forma mais moderna de formatação
  • O print() tem parâmetros úteis como sep e end
  • Sempre trate possíveis erros de conversão
